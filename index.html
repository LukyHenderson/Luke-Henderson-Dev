ADHomeD

Shopping List app integrated with Amazon, created both as a desktop app and a Website, the former using Python, CustomTkinter and MySQL and the latter using a combination of HTML, CSS, JavaScript and MySQL with Google Cloud SQL, Fast API and Google Cloud App Engine. This app makes use of the Amazon devlopment API to connect and retrieve information from the amazon product database. This app is currently a work in progress. The idea is supposed to help people with ADHD / other neurodivergencies to take control of their spending habits and buy only what they need, to help combat negative impulsive behaviours. The desktop app and webapp are cross compatible, sharing the same data on a MySQL database set up in Google Cloud SQL.
